# EV Control Pro | IL 🚗⚡
**מחשבון טעינה ותכנון נסיעה חכם לקהילת הרכב החשמלי בישראל.**

אפליקציית Web (PWA-Ready) שנועדה לתת מענה לבעלי רכבים חשמליים (בדגש על סדרת VW ID) שזקוקים לדיוק גבוה יותר מזה שמספקות אפליקציות היבואן המקומיות. הכלי מאפשר תכנון מדויק של זמני טעינה, עלויות ומרחק נסועה.

## 🌟 פיצ'רים מרכזיים
* **מחשבון מהיר (Quick Calc):** חישוב זמן טעינה נותר וזמן סיום משוער (ETA).
* **מתכנן יציאה (Departure Planner):** חישוב שעת החיבור האופטימלית כדי לצאת בזמן עם האחוז המבוקש.
* **חישוב שימוש (Usage Calc):** חישוב עלות נסיעה מבוססת kWh, השוואת טווח והערכת עלות לנסיעה.
* **מדידה בזמן אמת (Live Tracking):** סטופר המציג "זמן שחלף" (HH:MM:SS), אנרגיה שנכנסה (kWh) ועלות מצטברת (₪).
* **מנגנון נעילה (State Locking):** נעילת פרמטרים (Read-only) בזמן מדידה פעילה למניעת שגיאות.
* **שמירת העדפות משתמש:** שומר מותג, דגם, הספק ותעריף לטעינה הבאה.
* **מסד נתונים מובנה:** תמיכה בדגמי Tesla, VW, BYD, Xpeng ועוד המבוסס על **קיבולת נטו (Net Capacity)**.
* **UI/UX מודרני:** עיצוב Glassmorphism כהה, רספונסיבי לחלוטין ותומך RTL.

## 🛠 טכנולוגיה
* **Frontend:** HTML5, CSS3 (Tailwind CSS).
* **Logic:** Vanilla JavaScript (ES6+).
* **Architecture:** Decoupled data layer (`ev-database.js`).
* **Efficiency:** החישובים כוללים מקדם איבוד אנרגיה (Charging Efficiency) של 90%.
* **Storage:** העדפות משתמש נשמרות ב-LocalStorage לשימוש מהיר בעתיד.
